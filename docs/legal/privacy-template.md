## 1. はじめに

当社が運営する Stagecast (以下「本サービス」) は、利用者のプライバシーを尊重し、個人情報
保護法 (日本)・GDPR (欧州経済領域の利用者を対象とする場合)・その他関連法令を遵守して
個人情報を取り扱います。

## 2. 取得する情報の種類

### 2.1 利用者から直接取得する情報

| 種別   | 内容                                      | 取得タイミング |
| ------ | ----------------------------------------- | -------------- |
| 表示名 | 登壇者/モデレーターがステージ入室時に入力 | 入室時         |
| 音声   | マイクを ON にしたとき                    | 配信中         |
| 映像   | カメラを ON にしたとき                    | 配信中         |
| 画面   | 画面共有を ON にしたとき                  | 配信中         |
| Email  | 管理者の Cognito アカウント作成時         | サインアップ時 |

### 2.2 自動的に取得する情報

| 種別         | 内容                            | 取得目的                 |
| ------------ | ------------------------------- | ------------------------ |
| アクセスログ | IP アドレス、UA、リクエスト時刻 | セキュリティ・障害解析   |
| ブラウザ情報 | デバイス種別、解像度            | 配信品質の最適化         |
| Cookie 等    | 認証セッション                  | 管理コンソールの利用維持 |

## 3. 利用目的

取得した情報は、以下の目的で利用します:

1. ライブ配信の WebRTC 経由での中継 (LiveKit Server / Egress 経由)
2. 字幕の自動生成 (AWS Transcribe / Translate / Bedrock に音声を送信)
3. 録画ファイルの S3 保存と管理者へのダウンロード提供
4. YouTube Live への RTMP 送出 (映像・音声・字幕)
5. 障害解析・セキュリティ監視 (CloudWatch Logs)
6. 利用統計の集計 (個人を特定しない形)

## 4. 第三者提供

以下の場合を除き、利用者の同意なく個人情報を第三者に提供しません:

| 提供先                    | 提供する情報       | 目的                                             |
| ------------------------- | ------------------ | ------------------------------------------------ |
| Amazon Web Services, Inc. | 音声、映像、表示名 | サービスインフラ提供 (本サービスは AWS 上で稼働) |
| Google LLC (YouTube)      | 映像、音声、字幕   | YouTube Live 配信                                |
| 法令に基づく開示請求者    | 必要最小限         | 法令遵守                                         |

AWS および YouTube のプライバシーポリシーは、各社の公開規定をご参照ください。

## 5. データの保管期間

| 種別                   | 保管期間                        | 保管場所        |
| ---------------------- | ------------------------------- | --------------- |
| 録画ファイル           | 30 日 (実装上の TTL は別途設定) | Amazon S3       |
| 字幕ファイル (SRT/VTT) | 30 日                           | Amazon S3       |
| アクセスログ           | 1 ヶ月                          | CloudWatch Logs |
| 管理者アカウント       | 退会まで                        | Amazon Cognito  |

期間経過後は、当社の責任において安全に削除します。

## 6. 利用者の権利 (個人情報保護法 / GDPR)

利用者は、自身の個人情報について以下の権利を持ちます:

- アクセス権 (どんな情報が保管されているか開示請求)
- 訂正・削除請求権
- 利用停止請求権
- データポータビリティ (GDPR 該当者のみ)

権利行使は、お問い合わせ先にご連絡ください。原則 30 日以内に対応します。

## 7. Cookie とトラッキング

本サービスは以下の目的で Cookie / sessionStorage を利用します:

- Cognito 認証セッションの維持 (管理コンソール)
- 招待トークンの一時保持 (登壇者画面)

第三者トラッキング (広告等) は使用しません。

## 8. セキュリティ

当社は以下のセキュリティ対策を講じています:

- 通信の暗号化 (HTTPS / WSS / TLS)
- AWS Secrets Manager による機密情報の管理
- IAM 最小権限の原則
- CloudFront による DDoS 緩和
- AWS Cognito による多要素認証 (オプション)

## 9. 子どもの個人情報

本サービスは原則として 16 歳未満の利用を想定していません。16 歳未満の利用者が本サービスを
利用する場合は、保護者の同意が必要です。

## 10. プライバシーポリシーの変更

当社は、本ポリシーを予告なく変更することがあります。重要な変更がある場合は、本サービス上で
通知します。
