## 第1条 (適用)

1. 本利用規約 (以下「本規約」) は、当社が提供する Stagecast (以下「本サービス」) に
   関する一切の関係に適用されます。
2. 本サービスを利用する利用者 (登壇者、モデレーター、視聴者を含むがこれらに限られない)
   は、本規約に同意したものとみなします。

## 第2条 (サービス概要)

本サービスは、ライブ配信を YouTube Live 等の動画配信プラットフォームに送出するための
プラットフォームです。主な機能は以下のとおりです:

- 登壇者の WebRTC 経由でのステージ参加 (LiveKit)
- 字幕の自動生成 (Amazon Transcribe / Translate / Bedrock を利用)
- 録画の S3 保存と限定的なダウンロード
- YouTube Live への RTMP 送出

## 第3条 (アカウントと招待)

1. 本サービスの管理機能は、当社が Cognito で発行する管理者アカウント保持者のみが利用
   できます。
2. 登壇者・モデレーターは、管理者が発行する **招待 URL (JWT トークン)** によってステージに
   参加できます。招待 URL の有効期限はデフォルトで 12 時間です。
3. 利用者は、自身のアカウント情報・招待 URL を第三者に開示してはなりません。

## 第4条 (収集する情報)

本サービスの利用にあたって、以下の情報を収集する場合があります:

- 登壇者・モデレーター: 表示名、音声、映像、画面共有内容
- 管理者: Email アドレス (Cognito 認証用)
- システム: アクセスログ、IP アドレス、ブラウザ情報 (障害解析・セキュリティ目的)

詳細は [プライバシーポリシー](./privacy-template.md) をご参照ください。

## 第5条 (字幕生成と AI 利用)

1. 本サービスは登壇者の音声を Amazon Web Services (AWS) の Transcribe / Translate /
   Bedrock に送信し、字幕を自動生成します。
2. 字幕は配信中のオーバーレイ表示、YouTube Live のクローズドキャプション送出、配信後の
   SRT/VTT ファイルとして S3 に保存されます。
3. AWS のデータ取り扱いは、AWS の利用規約 / 各サービスの仕様に従います。

## 第6条 (録画)

1. 本サービスは LiveKit Egress によって配信内容を録画し、Amazon S3 に保存します。
2. 録画は管理者が当社の S3 バケットから 30 日間ダウンロード可能とします (実装上の TTL は
   別途設定)。
3. 利用者は録画されることに同意したものとみなします。

## 第7条 (YouTube Live への送出)

1. 本サービスは管理者が設定する YouTube ストリームキーを用いて、配信内容を YouTube Live
   に RTMP で送出します。
2. YouTube Live で公開された内容は、YouTube の利用規約・コミュニティガイドラインに従います。
3. 著作権侵害・名誉毀損等の責任は、配信を行った管理者・登壇者が負うものとします。

## 第8条 (禁止行為)

利用者は、以下の行為をしてはなりません:

- 法令または公序良俗に反する行為
- 第三者の著作権・知的財産権・プライバシー権を侵害する行為
- 本サービスのインフラに過度な負荷をかける行為
- 招待 URL を不正に転用・販売する行為
- 本サービスを介して第三者を誹謗中傷する行為

## 第9条 (本サービスの停止・変更)

1. 当社は、メンテナンス・障害・天災等の理由により、事前通告なく本サービスを停止する
   ことができます。
2. 当社は、本規約・本サービスの内容を予告なく変更できます。変更後の規約は本サービスに
   掲載した時点で効力を発します。

## 第10条 (免責)

1. 本サービスは「現状のまま」提供され、当社は配信品質、字幕の正確性、第三者プラットフォーム
   (YouTube 等) の挙動について保証しません。
2. 本サービスの利用に起因する損害について、当社の故意または重大な過失がある場合を除き、
   当社は責任を負いません。

## 第11条 (準拠法・管轄)

本規約は日本法に準拠し、本サービスに関する紛争は、当社の所在地を管轄する裁判所を第一審の
